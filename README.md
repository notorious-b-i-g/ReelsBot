# ReelsBot

Телеграм-бот для генерации русскоязычного контента для Instagram: Reels, посты и Stories на базе OpenAI GPT-4o.

## О проекте

ReelsBot — это ассистент-копирайтер в Telegram, который помогает создавать контент для блога: тексты постов, сценарии Reels и идеи для Stories. Пользователь заполняет короткую форму (ниша, цель, аудитория, формат, стиль), а бот формирует запрос к OpenAI GPT-4o и возвращает готовый результат.

Бот работает по принципу опроса (long polling) через aiogram. Для каждого пользователя ведётся история диалога и собственный системный промпт, которые хранятся в локальной базе SQLite — это позволяет учитывать контекст предыдущих сообщений. Доступ к боту ограничен allow-list: пользователь добавляется в список разрешённых по ссылке с ключом-приглашением.

## Стек

- **Python 3.11**
- **aiogram 3.4.1** — асинхронный фреймворк для Telegram Bot API
- **aiohttp** — HTTP-клиент для запросов к OpenAI и поддержки прокси
- **OpenAI Chat Completions API** (модель `gpt-4o`)
- **tiktoken** — подсчёт токенов и обрезка истории под лимит контекста
- **SQLite** — хранение истории диалогов, системных промптов и allow-list
- **python-dotenv** — загрузка конфигурации из `.env`
- Опциональный HTTP-прокси для доступа к OpenAI

## Возможности

- Генерация текстов постов по заполненной форме
- Создание сценариев и идей для Reels
- Подбор идей и концепций для Stories
- Стандартный режим свободного диалога с GPT-4o
- Хранение истории переписки по каждому пользователю (SQLite) с автоматической обрезкой по числу токенов
- Переключение и сброс системного промпта, очистка истории
- Контроль доступа через allow-list (вход по ссылке с ключом)
- Опциональная работа через HTTP-прокси (в том числе с авторизацией) для запросов к OpenAI
- Логирование в консоль и в файл `bot.log`

## Запуск

Требуется Python 3.11 и токен Telegram-бота, а также ключ OpenAI API.

```bash
# 1. Создайте и активируйте виртуальное окружение
python -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate

# 2. Установите зависимости
pip install -r requirements.txt

# 3. Подготовьте файл с переменными окружения
cp .env.example .env            # Windows: copy .env.example .env
# и заполните значения в .env

# 4. Запустите бота
python main.py
```

При первом запуске автоматически создаётся файл базы данных `database.db`.

## Переменные окружения

Конфигурация задаётся в файле `.env` (см. `.env.example`). Файл `.env` не попадает в git.

| Переменная | Обязательная | Описание |
|---|---|---|
| `OPENAI_API_KEY` | да | Ключ доступа к OpenAI API |
| `TELEGRAM_BOT_TOKEN` | да | Токен Telegram-бота (получается у @BotFather) |
| `PROXY_URL` | нет | URL HTTP-прокси для запросов к OpenAI. Если не задан — запросы идут напрямую |
| `PROXY_USER` | нет | Логин для авторизации на прокси |
| `PROXY_PASS` | нет | Пароль для авторизации на прокси |
